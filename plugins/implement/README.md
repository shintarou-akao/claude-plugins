# implement

planファイルに基づいて実装を実行し、チェックリストを更新しながら進捗を管理するプラグイン。

## 含まれるスキル

| スキル名 | コマンド | 説明 |
|---------|---------|------|
| implement | `/implement ~/.claude/plan/XXX.md` | planファイルに基づき実装・進捗管理 |

## 使い方

1. `/implement ~/.claude/plan/XXX.md` を実行
2. 実行対象のPRを選択（全PR or PR番号指定）
3. チェックリスト項目を順次実行し、完了時に `[x]` に更新
4. 検証セクションも実行し完了後に確認を受ける

## 動作の特徴

- PR内のタスクは確認なしで連続実行
- タスクや検証が失敗した場合は自律的に修正を試みる
- 全PR完了後にコミット/プッシュ/PR作成へのアクションを提案
