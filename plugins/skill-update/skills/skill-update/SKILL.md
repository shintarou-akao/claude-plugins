# スキル更新

会話中に得た学びやフィードバックをもとに、既存のスキルファイルを更新するワークフロー。

## ワークフロー

### 1. 更新対象の特定

引数または会話コンテキストから更新対象のスキルを特定。引数がない場合は直近の会話で使用したスキルを候補として提示。

### 2. 現在のスキル内容を確認

対象スキルを読み込む。

### 3. 更新内容の分析

会話から学んだ内容を分析し、更新ポイントを整理:

- 何が問題だったか
- どう改善するか
- 具体的な指摘は抽象化して他のケースでも流用できるようにする

### 4. 更新内容の確認

AskUserQuestionツールを使用して更新内容をユーザーに確認。

### 5. スキルファイルの更新

確認後、SKILL.mdを編集。
