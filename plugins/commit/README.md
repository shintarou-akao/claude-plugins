# commit

ステージされた変更から日本語の Conventional Commit メッセージを自動生成し、コミットを実行するプラグイン。

## 含まれるスキル

| スキル名 | コマンド | 説明 |
|---------|---------|------|
| commit | `/commit` | ステージ済み変更を分析し、日本語 Conventional Commit メッセージで自動コミット |

## 使い方

1. `git add` でコミットする変更をステージする
2. `/commit` を実行
3. 変更内容に基づいて最適なコミットメッセージが生成され、コミットが実行される

## コミットメッセージ形式

`type(scope): 説明`（日本語、50文字以内推奨）
