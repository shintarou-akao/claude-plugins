# spec-plan

仕様書から詳細な実装計画を自動生成し、planファイルとして出力するプラグイン。

## 含まれるスキル

| スキル名 | コマンド | 説明 |
|---------|---------|------|
| spec-plan | `/spec-plan path/to/spec.md` | 仕様書からPR単位の実装計画を生成 |

## 使い方

1. `/spec-plan path/to/spec.md` を実行
2. 仕様書を読み込み、コードベースを探索
3. PR単位で実装計画を構成し、`~/.claude/plan/` に出力する

## 計画の粒度

- 各PRは独立してレビュー・マージ可能
- 1PR = 1種類の修正
- 破壊的変更は別PRに分離
