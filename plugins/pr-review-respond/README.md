# pr-review-respond

PRのレビューコメントを取得し、指摘事項に対応してコミット・プッシュを行うプラグイン。

## 含まれるスキル

| スキル名 | コマンド | 説明 |
|---------|---------|------|
| pr-review-respond | `/pr-review-respond` | レビューコメント対応・コミット・プッシュ・返信 |

## 使い方

1. `/pr-review-respond` を実行（引数なし：現在のブランチのPRを自動検出）
2. レビューコメントが一覧表示され、妥当性が分析される
3. 対応するコメントを選択
4. コメントごとに修正・コミットを繰り返す
5. まとめてプッシュ
6. 対応内容をスレッドで返信（人間・AIツールで返信スタイルを切り替え）
